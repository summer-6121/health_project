# health_parent

#### 介绍
传智健康管理系统是一款应用于健康管理机构（慈铭、爱康国宾）的业务系统，实现健康管理机构工作内容可视化、会员管理专业化、健康评估数字化、健康干预流程化、知识库集成化，从而提高健康管理师的工作效率，加强与会员间的互动，增强管理者对健康管理机构运营情况的了解。
